Simple Google Drive manager.

Requirements:
    PHP version 5.6 or greater.

Installation:

 1. Open the Credentials page in the API Console.
 2.  Click Create credentials > OAuth client ID.
 3.  Complete the form. 
        - Set the application type to Web application. 
        - Specify authorized redirect URIs, like http://example.com/auth.
4. After creating your credentials, download the client_secrets.json file and securely store it in a Config folder.
